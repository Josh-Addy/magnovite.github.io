  • ThinkHI IDEATHON

    ThinkHI Ideathon welcomes you to send your boldest ideas based on Humanitarian Technologies for Sustainable Development. Let the marathon of ideas commence!

    Description and Rules

    In code you breathe, In code you sleep;
    Confidence you exude, In all things code;
    But where do you stand, when a problem you face;
    How quickly, an efficient solution can you make?
    Rules and Guidelines:
    1. The contest is open to anyone with a knack for programming.
    2. System and materials required will be provided.
    3.It is a team contest with each team having a maximum of two members from the same institution.
    4.Teams will be ranked as per the number of problems solved
    5.Ties will be broken by the total time for each user in ascending order of time.
    6.Total time is the sum of the time consumed for each problem solved. The time consumed for a solved problem is the time elapsed from the beginning of the contest to the submission of the first accepted run plus 20 penalty minutes for every previously rejected run for that problem (Wrong answer, Time limit exceeded, Runtime Error, Compilation Error). There is no time consumed for a problem that is not solved.
    7.The contest will consist of multiple rounds, with teams progressing in every round(3 rounds).
    8.Winner will be declared in the end of the event .
    9.Registration money is not refundable

  • RC Aero Modelling

    RC Aero Modelling is an event where a group of 2-3 have to design and create a flying model aircraft which successfully takes flight and clears the obstacle course and reach the finish point.

    Description and Rules

    DaVinci designed a flying machine,
    The first of it’s kind so man could fly;
    Can you replicate this wise man’s task,
    A model aircraft, can you craft?
    This aircraft must fly and evade all obstacles,
    Only then will a prize you claim.
    Rules and Guidelines:
    1.Participants must work in teams with a limit of 2-5 members per team.
    2.Bring the Model parts. Assembly of the parts can be done from the event venue.
    3.A team should use only one model throughout the competition.
    4.Models should be built from scratch and should not be purchased.
    5.In case of any damage, small modifications are allowed.
    6.Only electrical motors are allowed.
    7.Professionals are restricted from participation.
    8.Members of team may be from same or different college/school/clubs.
    9.If anyone found not following the above rules, they will be disqualified.
